# LaraForum is Laravel and Vue.js forum

### Install

###### 1. Clone repo 
 
`git clone https://github.com/milanarandjelovic/laraforum.git`

###### 2. Install dependencies
`composer install`

`npm install` or `yarn`

###### 3. Run the server
`php artisan serve`
